
Для целей проекта реализованы 4 класса:
- Main: основной класс, содержащий главный метод main и методы для выполнения задач по проекту
- Employee: который содержит информацию о ФИО, отделе и зарплате сотрудника
- EmployeeBook: класс, который содержит описание объекта с хранилищем сотрудников и методы по работе с ним
- EmployeeGenerator: вспомогательный класс для содания тестового набора данных
